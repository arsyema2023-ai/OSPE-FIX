⚙️ Konfigurasi Google Apps Script

Untuk melihat akses dari semua device, Anda perlu setup backend Google Sheets:

  1. Buka sheets.new → buat Spreadsheet baru
  2. Rename sheet jadi AccessLog
  3. Klik Extensions → Apps Script
  4. Copy kode dari file GOOGLE_SCRIPT.txt yang saya berikan
  5. Deploy sebagai Web App (Execute as: Me, Access: Anyone)
